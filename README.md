# fountn
Feline auto-hydration device.

## Design
### States
Error / Ready / Running
### Events
Touch + Release / In-range + out-of-range
Water flow on + off
Water level high + low
### Microprocessor
Spark Core / Photon for ease-of-update, logging and IoT
### Analog input
Water level sensor
Water flow
Touch / proximity sensor
### Analog output
Water pump
### Materials
Water bowl + reserve
Touch capacitive mat / proximity detector
Tubing
